![Book Manager](https://cdn.rawgit.com/bdTechies/cdn/ae91723d02f6e5ba222da5a62315e628ff55df3e/images/book-manager-welcome-screen.png)

### Install & Run

```bash
$ git clone git@github.com:bdTechies/book-manager.git
```

```bash
$ npm install 
```

```bash
$ npm start
```
